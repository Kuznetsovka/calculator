# Программа подбора увлажнителей

## Назначение приложения

Снижение временных затрат на получение коммерческих предложений на пароувлажнители

## Используемые технологии

* Java
* Maven
* Spring (Spring Boot, Spring MVC)
* Hibernate
* Библиотека для миграции баз данных Flyway
* Библиотеки для создания PDF-файлов [Apache PDFBox](https://pdfbox.apache.org) и [Boxable](http://dhorions.github.io/boxable)
* JUnit5

## Интересные особенности программы

* Выдача нескольких подходящих увлажнителей, из которых клиент может выбрать заинтересовавший
* Отправка PDF-файла на почту клиента и почту консультанта
